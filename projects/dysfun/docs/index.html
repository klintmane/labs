Monads

Functors implementing a 'join' and 'chain' method.

Maybe

Useful for abstracting error-handling, handles null/undefined values by short circuiting.

Maybe
Parameters
val (any)
Example
// If someFunction below returns null, Maybe.map will short-circuit/return
// {value: null} and will not run the next map in the chain
var greeting = D.Maybe.of("Bob")
                      .map(someFunction)
                      .map((x) => "Hello there, " + x);

// Without Maybe we would usually manually add error handling
if (result === null) {
    return null
} else {
    doSomething(result)
}

Functors

Pointed Functors: Objects implementing a 'map' and 'of' method.

Either

Like Maybe, but seperates failure and success into Left and Right objects, where Left can hold information on the failure, unlike Maybe's null value.

Either
Example
var nameEither = (x) => x !== undefined ? Either.Right.of(x) : Either.Left.of("Error: Name was not provided!");

var greeting = nameEither("Bob").map((x) => "Hello there " + x);
console.log(greeting); //-> returns Right {value: "Hello there Bob"}

var greetingUndefined = nameEither().map((x) => "Hello there " + x);
console.log(greetingUndefined);//-> returns Left {value: "Error: Name was not provided!"}

General

General High Order Functions.

unless

Runs a given function if the given predicate is false.

unless(fn: function, predicate: boolean): undefined
Parameters
fn (function) Function to be run if predicate is false.
predicate (boolean) Either boolean or returns true/false.
Returns
undefined: Runs fn if predicate is false.
Example
// outputs "Less than 10!" to the console if given x < 10
unless(x > 10, () => console.log("Less than 10!"))

times

Runs a given function a given number of times.

times(number: number, fn: function): undefined
Parameters
number (number) Number of times fn should be run.
fn (function) (time) - Function to be run number in total.
Returns
undefined: Runs fn a number of times.
Example
// outputs to the console 10 times
times(10, (x) => console.log(`Printing for the ${x} time`))

forEach

Traverses a given array and runs the given function with each array item.

forEach(arr: Array, fn: function): undefined
Parameters
arr (Array) Array to be traversed.
fn (function) (item) - Function to be applied to each array item.
Returns
undefined: Runs fn with each array item as an argument.
Example
// outputs each array item to the console
forEach([1, 2, 3], (a) => console.log(a))

forEachObj

Traverses a given object and runs the given function with each object property.

forEachObj(obj: Object, fn: function): undefined
Parameters
obj (Object) Object to be traversed.
fn (function) (key, val) - Function to be applied to each object property.
Returns
undefined: Runs fn with each key-value pair as arguments.
Example
// outputs each key-value pair to the console as "key:value"
forEachObj({x:0, y:1}, (k,v) => console.log(k + ":" + v))

every

Checks if a given function returns true for every item of a given array.

every(arr: Array, fn: function): boolean
Parameters
arr (Array) Array , the items of which will be evaluated.
fn (function) Function to evaluate each item.
Returns
boolean: true/false.
Example
// returns true
every([7, 6, 5], greaterThanFour)

sortBy

Returns a sorting function in ascending order for the object property, to be passed to Array.sort

sortBy(prop: any): function
Parameters
prop (any) Property of the object.
Returns
function: The sorting function to be passed to Array.sort.
Example
// returns persons array, sorted by age in ascending order
let persons = [{name: "Joe", age: 32}, {name: "Al", age: 21}]
persons.sort(sortBy("age"))

tap

Returns a function with closure over a given value, which outputs the value and executes a given function with the value.

tap(val: any): function
Parameters
val (any) A parameter of an unspecified type.
Returns
function: A function, which outputs the value and executes a given function with the value.
Example
// traverses array ouputting the current item and running the given function (output "Value is x")
forEach(arr, (x) => tap(x)(() => console.log(`Value is ${x}`)))

Array Utils

Functions helping with array operations.

filter

Traverses a given array, runs the given function with each array item and returns an array with the new items for which fn is true.

filter(arr: Array, fn: function): Array
Parameters
arr (Array) Array to be traversed.
fn (function) (item) - Function to be applied to each array item.
Returns
Array: New array where fn is true for each item.
Example
// returns [1, 2]
filter([1, 2, 3], (a) => a<3)

map

Traverses a given array, runs the given function with each array item and returns an array with the new items.

map(arr: Array, fn: function): Array
Parameters
arr (Array) Array to be traversed.
fn (function) (item) - Function to be applied to each array item.
Returns
Array: New array where each item is fn(arr.item).
Example
// returns [2, 4, 6]
map([1, 2, 3], (a) => a*2)

reduce

Reduces a given array into a single value from applying a given function to the accumulator and each item.

reduce(arr: Array, fn: function, init: any?): Array
Parameters
arr (Array) Array to be reduced.
fn (function) Reducing function.
init (any?) Initial value of the accumulator.
Returns
Array: Containing the value of the reduced array.
Example
// returns the sum [6]
reduce([1, 2, 3], (acc, item) => acc + item)
// returns the product [8]
reduce([4, 2], (acc, item) => acc * item)
// returns [{kiwi: 7}]
arr = [{kiwi: 2, pear: 5}, {kiwi: 5, pear: 0}]
reduce(arr, (acc, item) => {return {kiwi: acc.kiwi + item.kiwi}}, {kiwi: 0})

concatAll

Flattens a one-level deep nested array.

concatAll(arr: Array): Array
Parameters
arr (Array) Array to be flattened.
Returns
Array: Flattened array.
Example
// returns ["A", "B"]
concatAll([["A"], ["B"]])

zip

Zips two given arrays together.

zip(arrL: Array, arrR: Array, fn: function): Array
Parameters
arrL (Array) The first array.
arrR (Array) The second array.
fn (function) (itemL, itemR) - Function for zipping items together.
Returns
Array: New array resulting from the zip.
Example
// returns [8, 7]
zip([5, 6, 8], [3, 1], (x, y) => x + y)

Currying/Partial Application

curry

Takes a function with an arity of n and returns n nested unary functions.

curry(fn: function): function
Parameters
fn (function) Function with n arguments.
Returns
function: Nested functions taking one argument.
Example
add = curry((x, y, z) => x + y + z)
add(5)(8)(3) //would otherwise be add(5, 8, 3)

partial

Takes a function fn and arguments argsP, some undefined, returns a function taking the undefined arguments, that when called runs fn with the arguments combined.

partial(fn: function, argsP: any): function
Parameters
fn (function) Function
argsP (any) Partial supplied arguments.
Returns
function: Function taking the remaining undefined arguments.
Example
add = (x, y) => x + y
addTwo = partial(add, 2, undefined)
addTwo(5) // returns 7

Composition/Piping

compose

Takes functions f1, f2, ..., fn and returns a single function accepting arg, which calls f1(...(fn(arg))).

compose(fns: ...function, arg: any?): function
Parameters
fns (...function) Functions to be composed.
arg (any?) Argument to be passed to the returned function.
Returns
function: Accepting arg.
Example
square = (x) => x*x
double = (x) => x*2
doubleSquare = compose(double, square)
doubleSquare(4) //-> returns 32

pipe

Takes functions f1, f2, ..., fn and returns a single function accepting arg, which calls fn(...(f1(arg))).

pipe(fns: ...function, arg: any?): function
Parameters
fns (...function) Functions to be piped.
arg (any?) Argument to be passed to the returned function.
Returns
function: Accepting arg.
Example
square = (x) => x*x
double = (x) => x*2
squareDouble = pipe(double, square)
squareDouble(4) //-> returns 64
